Stored in localStorage. Client-side keys are exposed — use cautiously.

Keyboard shortcuts are fixed for now. Fudelre update will allow remapping.

  • Enter — send message
  • Shift + Enter — newline
  • ↑/↓ — previous/next prompt
  • Ctrl/⌘ + \ — toggle split
  • Ctrl/⌘ + ←/→ — focus pane
  • Ctrl/⌘ + B — toggle sidebar
  • Esc — close popover/modal

BEEF is a minimal, dark, grid-aesthetic AI chat UI built with plain HTML/CSS/JS. Icons are Material Symbols; fonts come from Google Fonts.

Gemini requests are performed directly from the browser. Use restricted keys or a proxy for production environments.
